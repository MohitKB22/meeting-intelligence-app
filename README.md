## ✨ Overview

AI Meeting Assistant helps teams capture discussions and automatically convert them into:

* ✅ Action Items
* ✅ Decisions
* ✅ Risks
* ✅ Meeting Summaries
* ✅ Follow-up Emails
* ✅ Meeting Intelligence & Insights

No frameworks. No dependencies. Just HTML, CSS, JavaScript, and Claude AI.

---

## 🎯 Key Features

### 📝 Smart Meeting Notes

Capture notes in real time and let AI automatically classify them into:

* Decisions
* Action Items
* Risks
* Questions
* Discussions

### ✅ Action Tracker

Track ownership, priority, due dates, and status.

### 📋 Decision Log

Maintain a structured record of meeting decisions and reasoning.

### ⚠️ Risk Register

Identify potential blockers and mitigation plans before they become issues.

### 🤖 AI-Powered Outputs

Generate:

* Executive Summaries
* Detailed Meeting Minutes
* Follow-up Emails
* Formal Meeting Reports

### 📊 Meeting Intelligence

Get insights on:

* Team sentiment
* Engagement levels
* Stakeholder alignment
* Strategic recommendations
* Improvement opportunities

## 🚀 Quick Start

```bash
git clone https://github.com/yourusername/meeting-assistant-ai.git

cd meeting-assistant-ai
```

Open:

```bash
index.html
```

Or run a local server:

```bash
python3 -m http.server 8080
```
